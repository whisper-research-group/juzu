juzu: the main program of the "juzu" project.

(init: initializes "../data-dir")
julius: an internal engine of "juzu" used for speech segmentation.
fuzuki: a program controling "julius" in "juzu."
nuzume: the new version of "uzume."
kazune: a program overtonizing the output from "nuzume" in "juzu."
kurabe: compairs the data.
finale: gathers the data and put into final-outputs.
(term: terminates whole the program.)
